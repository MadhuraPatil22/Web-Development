Following are the different types of tags used in html :
The 'b' HTML tag is used to make the font BOLD .
This HTML tag defines the text with strong importance.Content in this is typically displayed in bold
The'i' HTML tag is used to make the font in Italic.
The 'u' HTML tag is used to make the font in Undereline.
This tag is used to make text emphasized
The 'small' HTML tag is used to define the smaller text.
The 'big' HTML tag is used to make text big.
The 'mark' HTML tag defines the text that should be highlighted or marked
The 'del' HTML tag is used to delete the content .My favourite colour is RedPurple.
The 'ins' HTML tag is used to insert the element.My favourite colour is Red Pink
