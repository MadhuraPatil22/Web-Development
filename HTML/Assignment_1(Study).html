Study of Web and Technology

This practical introduces the fundamental components of web development and the technologies that power modern websites and applications. It helps students understand how websites are built, styled, and made interactive.

Topics Covered

  • HTML (HyperText Markup Language): Used to create the structure and content of web pages using elements like headings, paragraphs, links, tables, forms, etc.
  • CSS (Cascading Style Sheets): Used for designing and improving the look and layout of web pages including colors, fonts, spacing, and responsive design.
  • JavaScript: A scripting language that enables interactivity such as form validation, animations, dynamic content updates, and more.
  • Web Browsers: Applications like Chrome, Firefox, and Edge that interpret HTML, CSS, and JavaScript to display web content.
  • Client-Server Architecture: Describes how web clients (browsers) request data from web servers and how responses are delivered using protocols like HTTP/HTTPS.
  • Web Hosting: Making a website available on the internet by uploading it to a web server using services like GitHub Pages, Netlify, or traditional hosting platforms.
  • Domain Names: Human-friendly addresses (e.g., www.example.com) used to access websites instead of IP addresses.
